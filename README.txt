# Vixed Requests Plugin

QGIS processing plugin to generate offline requests for data
to be served via email.

Generated requests are supposed to be sent as-is to service email address.

## Installation
* Download plugin as zip file
* Enter QGIS "Plugins" dialog
* Choose "Install from ZIP"

Plugin should be available in the "Processing" panel, as well as in the "Plugins>Vixed" and in the plugins toolbar.

Currently the following products can be requested:
* Sentinel-1 satellite radar maps of sea ice (SAR)
* Chlorophyll-A maps based on Sentinel-3 products
